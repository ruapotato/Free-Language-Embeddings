How It Works (V11)

The model compresses text into 32 concept slots (each a 32-dimensional vector, 1024 dims total). The encoder reads text and produces these concept vectors. A non-autoregressive parallel decoder reconstructs the original text from the vectors alone — all tokens decoded simultaneously, no teacher forcing.

Key insight: Geometry (paraphrase similarity, clustering, word-order sensitivity) emerges naturally from reconstruction with a parallel decoder. No explicit geometry losses are needed. V10 probing at 50K steps showed analogies 0.74–0.92 with zero geometry training.

V11 trains on diverse data: prose, code, math, technical docs, conversations, RFCs, man pages, kernel source, arxiv papers, StackExchange, and more (~8.4M texts). Sentences/paragraphs are chunked into full ideas.

Training Setup

Architecture 54.3M params, 32 slots × 32 dims = 1024-dim bottleneck
Schedule 300K steps, cosine LR 3e-4 → 1e-5, batch size 64
Dynamic Sampling Oversamples weak length buckets (short/medium/long). Weight ∝ (1−em)^α with floor. Shifts training focus to lengths that aren’t solved yet.
Loss Pure reconstruction cross-entropy. No geometry losses, no phases, no gates.

Metrics

recon — Reconstruction cross-entropy loss. The only training signal. Lower is better.
token_acc — Per-token accuracy. What fraction of tokens does the decoder get right?
exact_match — Full-sentence exact match. The entire output matches the input word-for-word.
em_ema — Exponential moving average of exact match (decay=0.9). Smoothed training signal.
short/med/long — Per-bucket metrics. Short = 1–4 tokens, medium = 5–8, long = 9+.
dynamic weights — Sampling weights for each bucket. Higher weight = more training focus.
lr — Learning rate. Cosine schedule from 3e-4 peak down to 1e-5 minimum.

Diagnostic Sentences

[OK] — Perfectly reconstructed. Green = working.
[DIFF] (X%) — X% token overlap. Shows input → output to track what the model gets wrong.
Diagnostics include prose, code (def fibonacci...), math (derivative of x squared...), and logic (if all dogs are animals...).
